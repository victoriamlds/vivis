# 📚 Livraria Online

Projeto de uma livraria online desenvolvido com o objetivo de praticar desenvolvimento web, organização de código e boas práticas de programação.

---

## 🚀 Funcionalidades

* 📖 Visualização de livros disponíveis
* 🔍 Busca de livros por título ou categoria
* 🛒 Adição de livros ao carrinho
* 💰 Simulação de compra
* 📋 Exibição de detalhes dos livros

---
